教程准备:一台能联网的电脑或者手机、一双爪子、一个脑子
就直接就进入主题啦

电脑教程

  1. 访问 https://218.22.71.210:4433 [电信用户] 或 https://111.38.190.131:4433 [移动联通用户]（只能在外网访问，校园网内不能访问），提示证书有问题，不管它，点击“继续浏览此网站”
  2. 输入自己的学号和密码，点击登录
  3. 点击下载按钮下载山石VPN(这个将用于访问学校内网)
  4. 打开后输入服务器地址:
标题 填入内容
电信用户服务器地址 218.22.71.210
移动联通服务器地址 111.38.190.131
端口号 4433
用户号 学号
密码 自己的密码
  1. 链接成功后托盘会显示:
  2. 然后登录门户网站就可以查成绩了
